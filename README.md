# Sistema Bancário em Python 💰🐍

Projeto de estudo para praticar lógica de programação, organização de código e operações bancárias básicas. 🧠📚

## Funcionalidades ✨
- Depositar valores 💵
- Sacar valores 🏧
- Consultar extrato 📄
- Sair do sistema 🚪

## Objetivo 🎯
Simular um sistema bancário simples em Python para treinar estruturas condicionais, laços de repetição e manipulação de dados.
